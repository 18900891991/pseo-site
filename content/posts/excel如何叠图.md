# Excel 叠图（Overlay Charts）教程

在数据可视化中，叠图（Overlay Chart）可以让你把多组数据放在同一坐标系里，便于直接比较或展示两种不同尺度的数据。本文将从**为何需要叠图**讲起，给出**三种常用解决方案**，并回答**最常见的疑问**，帮助你在 Excel 中轻松实现叠图。

---

## 1. 为什么要做叠图？

| 场景 | 需求 | 叠图的优势 |
|------|------|------------|
| 销售与利润 | 同时展示销量与利润，销量大、利润小 | 直接对比，节省空间 |
| 预算与实际 | 同时查看预算与实际支出 | 直观发现偏差 |
| 目标与达成 | 目标值与实际值 | 及时评估进度 |
| 多维度 | 同时展示不同指标 | 统一视图，减少图表数 |

> **小贴士**：叠图不等于“堆叠图”。堆叠图是把各系列的值相加，叠图是把系列放在同一坐标轴上，彼此叠加。

---

## 2. 方案一：组合图（Combination Chart）

组合图是最常用的叠图方式，允许你把柱形图、折线图、面积图等多种图形叠加在同一坐标轴。

### 步骤

1. **准备数据**  
   | 月份 | 销售额 | 利润率 |
   |------|--------|--------|
   | 1月 | 12000 | 8% |
   | 2月 | 15000 | 10% |
   | 3月 | 11000 | 7% |

2. **插入图表**  
   - 选中整个数据区域  
   - `插入` → `图表` → `柱形图`（先选一个柱形图）

3. **设置组合图**  
   - 选中图表，右键 → `更改图表类型`  
   - 在左侧选择 `组合`，为不同系列分配图表类型（如：销售额 → 柱形，利润率 → 折线）  
   - 确认 `主轴` 和 `次轴` 的选择（如利润率使用次轴，以便比例匹配）

4. **调整格式**  
   - 右键系列 → `设置系列格式` → `填充`，为柱形设置透明度（如 50%），让折线能清晰可见  
   - 调整轴标签、图例、标题等

5. **完成**  

> **提示**：如果系列数较多，建议把不需要的系列隐藏，以避免图表过于拥挤。

---

## 3. 方案二：叠加系列（Overlay Series）

如果你想把 *同一数据类型* 的多组数据叠加在同一图表里（如多条折线），可采用“叠加系列”方法。

### 步骤

1. **准备数据**  
   | 日期 | 方案A | 方案B | 方案C |
   |------|------|------|------|
   | 1/1 | 20 | 25 | 18 |
   | 1/2 | 30 | 28 | 24 |
   | 1/3 | 25 | 35 | 22 |

2. **插入折线图**  
   - 选中所有列  
   - `插入` → `折线图` → `折线`

3. **同轴叠加**  
   - 默认所有系列已在同一坐标轴；如果不想显示所有图例，可以右键系列 → `设置系列格式` → `系列选项` → `系列在同一图表`（保持默认）

4. **美化**  
   - 通过图表工具 `格式` → `系列` → `颜色` 或 `线条样式` 区分各方案  
   - 为每条线添加数据标签（可选）

5. **完成**  

> **小技巧**：如果你想让某条线更突出，可将其线宽设大或使用不同的标记形状。

---

## 4. 方案三：使用图层与透明度（Layered Charts with Opacity）

当你需要将两种完全不同的图表（如柱形+面积）叠加在同一坐标轴时，可以通过图层和透明度实现。

### 步骤

1. **准备两套数据**  
   - 数据A（柱形）  
   - 数据B（面积）

2. **插入第一个图表**  
   - 选中数据A → `插入` → `柱形图`

3. **复制第二套数据并粘贴为图表**  
   - 选中数据B → `插入` → `面积图`  
   - 复制得到的面积图

4. **叠加图表**  
   - 将第二个图表直接拖放到第一个图表上方，使两者位置完全重合  
   - 右键第二个图表 → `复制到同一工作表` → `粘贴为图表`  
   - Excel 会自动把两张图表合并为一个组合图

5. **调整透明度**  
   - 对柱形图系列 → `设置系列格式` → `填充` → `填充透明度`（例如 30%）  
   - 对面积图系列 → `设置系列格式` → `填充透明度`（例如 70%）

6. **同步坐标轴**  
   - 如果两组数据尺度不同，右键系列 → `设置系列格式` → `次坐标轴`，并根据需要调整轴范围

7. **完成**  

> **建议**：在叠加前，先在 Excel 里预览两张图表单独显示，确保颜色、线型不会混淆。

## 5. 常见问题解答（FAQ）

| 问题 | 解决方案 |
|------|----------|
| **Q1. 叠图后数据标签重叠怎么办？** | 1）打开 `图表工具` → `布局` → `数据标签` → `更多标签选项`；<br>2）调整 `标签位置` 为 `外侧` 或 `内部左侧`；<br>3）对冲突的标签手动拖动。 |
| **Q2. 如何在叠图中显示两种不同单位的轴？** | 1）在 `更改图表类型` 里为某系列选择 `次坐标轴`；<br>2）右键次轴 → `设置坐标轴格式` → 选择合适的刻度与单位。 |
| **Q3. 叠图后图例显示不完整？** | 1）确保所有系列都在图表内；<br>2）右键图例 → `设置图例格式` → 调整 `图例框` 的尺寸；<br>3）若仍不行，可手动添加文本框作为图例。 |
| **Q4. 叠图后某系列不显示，怎么办？** | 1）检查该系列是否被错误地设置为 `隐藏`；<br>2）右键图表 → `选择数据` → 确认该系列已勾选；<br>3）检查系列颜色与背景是否冲突。 |
| **Q5. 如何在叠图中保留原始数据的趋势线？** | 1）右键系列 → `添加趋势线`；<br>2）在 `趋势线选项` 中选择 `线性/指数/多项式`；<br>3）调节颜色与线宽。 |

---

## 6. 小结

- **叠图**是将多组数据放在同一坐标轴上的有效手段。  
- **组合图**、**叠加系列**、**图层透明度**是三种最常用的实现方式，适用于不同类型的数据。  
- 通过合理的轴设置、透明度、图例和数据标签，你可以让叠图清晰、易读。
